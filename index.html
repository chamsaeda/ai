선형회귀

직선으로 데이터의 경향 찾기

1

직선의 방정식 → y = ax + b

2

최적의 직선 → 데이터와 가장 가까운 선

3

활용 예시 → 키로 몸무게 예측하기

💡 핵심 개념: 직선, 기울기, 오차 최소화

다중선형회귀

여러 요인을 고려한 예측

1

다변수 방정식 → y = a₁x₁ + a₂x₂ + ... + b

2

여러 특성 → 복합적인 요인 반영

3

활용 예시 → 키, 나이로 몸무게 예측

💡 핵심 개념: 다변수, 복합 요인, 평면/초평면

로지스틱회귀

Yes/No 분류 문제 해결하기

1

확률 예측 → 0과 1 사이의 값

2

S자 곡선 → 시그모이드 함수

3

활용 예시 → 합격/불합격 예측

💡 핵심 개념: 확률, 이진 분류, 시그모이드

K-최근접 이웃 (KNN)

가까운 이웃들의 투표로 결정

1

거리 계산 → 가장 가까운 이웃 찾기

2

다수결 투표 → K개 이웃의 의견 종합

3

활용 예시 → 영화 장르 분류

💡 핵심 개념: 거리, K값, 다수결

서포트 벡터 머신 (SVM)

최적의 경계선으로 분류하기

1

결정 경계 → 집단을 나누는 선

2

마진 최대화 → 가장 여유로운 경계

3

활용 예시 → 이메일 스팸 필터

💡 핵심 개념: 초평면, 마진, 서포트 벡터

2

이상치 탐지 → 정상 범위 벗어난 데이터

3

활용 예시 → 불량품 검출

📖 회귀 알고리즘

숫자를 예측하는 알고리즘입니다. 키로 몸무게 예측하기, 공부 시간으로 성적 예측하기 같은 문제를 해결합니다.

🎯 분류 알고리즘

범주를 구분하는 알고리즘입니다. 사진 속 동물이 고양이인지 강아지인지, 이메일이 스팸인지 아닌지 판단합니다.

🎨 군집화 알고리즘

정답 없이 비슷한 데이터끼리 묶는 알고리즘입니다. 고객을 비슷한 소비 패턴끼리 그룹으로 나눌 수 있습니다.

💪 학습 방법

각 알고리즘 페이지에서 직접 데이터를 입력하고 결과를 확인해보세요. 반복 학습이 가장 좋은 이해 방법입니다!
